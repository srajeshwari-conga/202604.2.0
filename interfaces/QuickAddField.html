Description

Describes a field entry for the quoteFields or orderFields inputs on the QuickAddComponent.

Index

Properties
  • Optional errorMsg
  • field
  • Optional lookupOptions
  • Optional required

Properties

errorMsg
errorMsg: string
Type : string
Optional

Inline validation message displayed below the field when it has no value. Accepts a plain string or an i18n translation key.
field
field: string
Type : string

The API name of the field on the Quote or Order object (e.g. 'PartnerAccount', 'Description').
lookupOptions
lookupOptions: LookupOptions
Type : LookupOptions
Optional

Custom lookup configuration controlling which records are fetched and how they are displayed. When omitted, the component uses the default lookup behaviour for the field. See LookupOptions for more details on the structure and fields required.
required
required: boolean
Type : boolean
Optional

When true, the Place Quote / Place Order button stays disabled until this field has a non-empty value. Defaults to false.
